# This is my Momoir

This is the personal blog of Jen Vega where she documents her journey as a Mom of 2 princesses. Built with Nuxt 4, Tailwind CSS, and Decap CMS.

## Getting Started

### Requirements

- Node v18+
- Nuxt 4
- Tailwind CSS

### Development

```bash
npm install
npm run dev
```

### Content Management

Blog posts are managed through [Decap CMS](https://decapcms.org/) and stored as Markdown files in `content/blog/`.
